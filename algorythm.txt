1)	Берем ссылку с сайта:
2)	Ебашим в баш: (git clone *ссылка*)
3)	Не забываем про команды, если надо:
		pwd – где мы
		cd – пройти в папку
		“cd –“ – вернуться назад
		ls – смотрим что в папке
		touch – создать файл
		mkdir – создать папку
		rm – удалить
4)	Потом что-либо делаем по проекту например
5)	Когда что-то сделали и надо добавить пишем (git add --all)
6)	Пакуем это в коммит и пишем (git commit –m ‘комментарий’)
7)	Потом если все ок пушим (git push)

8) 	Если было внесено изменение, и версия проекта на компе по сути устарела и в ней отсутствуют изменения, котрые уже есть в мейн ветке, то нужно прописать комманду (git pull)
9) 	Показать какие ветки есть в проекте (git branch)
10) 	Чтобы создать ветку и переключиться на нее пишем (git checkout -b название ветки)
11) 	Чтобы переключаться между ветками команда (git checkout название ветки)
12) 	Для создания новой ветки команда (git push --set-upstream origin название ветки)
13)	Чтобы удалить ветку со всем ее содержимым команда (git branch -d название ветки)
